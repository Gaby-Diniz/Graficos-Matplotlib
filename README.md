# Graficos-Matplotlib
Gráficos para representar cenários.
1. Crie um gráfico de linha mostrando a evolução da quantidade de um produto específico ao longo de uma semana. Imagine dados como dias = [1, 2, 3, 4, 5, 6, 7] e estoque = [100, 95, 110, 105, 120, 115, 130].

2. Desenvolva um gráfico de barras para comparar as quantidades em estoque de diferentes produtos. Utilize dados como produtos = ['Teclado', 'Mouse', 'Monitor', 'Webcam'] e quantidades = [50, 75, 30, 60].

3. Gere um gráfico de pizza para visualizar a proporção do valor total de estoque por categoria de produto. Considere categorias = ['Eletrônicos', 'Vestuário', 'Alimentos'] e valores = [15000, 8000, 5000].

4. Faça um gráfico de dispersão para analisar a relação entre o preço unitário e a quantidade em estoque de diversos itens. Dados de exemplo: precos = [50, 120, 300, 80, 20] e estoque = [80, 25, 10, 70, 150].
